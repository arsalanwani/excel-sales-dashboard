# Sales Dashboard – Excel Project

## Project Overview
This project analyzes sales data using Excel to create an interactive dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- KPI Cards
- Data Cleaning

## Dashboard Features
- Total Sales KPI
- Profit Analysis
- Category Performance
- Region-wise Sales

## Dataset
Sample sales dataset used for dashboard creation.
